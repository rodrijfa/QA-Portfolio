# BUG-006 — Campo Last Name no acepta input en checkout con error_user

## Descripción
Con error_user, el campo Last Name del formulario de checkout no permite 
escribir ningún valor. El campo aparece visualmente disponible pero 
no registra ningún input del usuario.

## Pasos para reproducir
1. Login con error_user / secret_sauce
2. Agregar cualquier producto al carrito
3. Ir al carrito y click en "Checkout"
4. Intentar escribir cualquier valor en el campo Last Name

## Resultado esperado
El campo debería aceptar texto como cualquier campo de formulario estándar.

## Resultado obtenido
El campo no registra ningún input. Permanece vacío sin importar 
lo que el usuario escriba.

## Impacto
El usuario no puede completar correctamente sus datos de envío, 
comprometiendo la integridad de la información del pedido.
