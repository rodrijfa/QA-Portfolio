## Description
With error_user, the "Finish" button on the order summary screen 
does not respond to clicks, preventing the purchase process 
from being completed.

## Steps to Reproduce
1. Log in with error_user / secret_sauce
2. Add any product to the cart
3. Go to cart and click "Checkout"
4. Fill in First Name and Postal Code (Last Name does not accept input)
5. Click "Continue"
6. Review the summary and click "Finish"

## Expected Result
The purchase should be completed and display the confirmation screen 
with the message "Thank you for your order".

## Actual Result
The "Finish" button does not respond. The screen remains on the 
order summary with no action or error message.

## Impact
The complete purchase flow cannot be completed with this user.
This is the highest severity bug identified in this testing session,
as it blocks the main functionality of the application.
