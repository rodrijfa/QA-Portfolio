## Description
With error_user, the Last Name field in the checkout form does not 
accept any input. The field appears visually available but does not 
register any user input.

## Steps to Reproduce
1. Log in with error_user / secret_sauce
2. Add any product to the cart
3. Go to cart and click "Checkout"
4. Attempt to type any value in the Last Name field

## Expected Result
The field should accept text like any standard form field.

## Actual Result
The field does not register any input. It remains empty regardless 
of what the user types.

## Impact
The user cannot correctly complete their shipping information, 
compromising the integrity of the order data.
